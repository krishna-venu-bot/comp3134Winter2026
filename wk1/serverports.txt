1. What is the definition of server port?
A server port is a communication endpoint used by computers to send and receive network traffic for specific services.

2. What is server port range?
Server ports range from 0 to 65535.

3. How many bits does the number above represent?
The number 65535 represents 16 bits.

4. State 10 common server ports:
- 21: FTP
- 22: SSH
- 23: Telnet
- 25: SMTP
- 53: DNS
- 80: HTTP
- 110: POP3
- 143: IMAP
- 443: HTTPS
- 3389: RDP
